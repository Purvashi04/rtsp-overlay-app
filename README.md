### RTSP Overlay Application

This project is a web-based application that plays a live video stream and allows users to add and manage text overlays on top of the video.

---

### DEMO VIDEO LINK

-> https://drive.google.com/file/d/1axQiUSHoKHqKI3DPfPEGo89goGiOmDOb/view?usp=drive_link

 ### Features 
 
-Video playback using a streaming URL
-Add multiple text overlays
-Drag and reposition overlays
-Simple and lightweight frontend
-Backend-ready for Flask and MongoDB

---
### Technologies Used

> HTML, CSS, JavaScript
> Flask (Backend)
> MongoDB
> HLS / RTSP-compatible video stream

---
### How to Run

1. Open index.html in a browser or using Live Server.
2. Enter a valid stream URL.
3. Click Play.
4. Add overlay text and drag it on the video.
