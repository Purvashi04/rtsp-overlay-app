### RTSP Overlay Application

This project is a web-based application that plays a live video stream and allows users to add and manage text overlays on top of the video.

---

### DEMO VIDEO LINK

-> https://drive.google.com/file/d/1axQiUSHoKHqKI3DPfPEGo89goGiOmDOb/view?usp=drive_link

 ### Features 
 
-Video playback using a streaming URL <br/>
-Add multiple text overlays <br/>
-Drag and reposition overlays <br/>
-Simple and lightweight frontend <br/>
-Backend-ready for Flask and MongoDB <br/>

---
### Technologies Used

> HTML, CSS, JavaScript <br/>
> Flask (Backend) <br/>
> MongoDB <br/>
> HLS / RTSP-compatible video stream <br/>

---
### How to Run

1. Open index.html in a browser or using Live Server. <br/>
2. Enter a valid stream URL. <br/>
3. Click Play.<br/>
4. Add overlay text and drag it on the video.<br/>
